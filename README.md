# Hyeon's photofolio

이 포토폴리오는 반응형 페이지로 구성되어 있습니다. 스마트폰 컨셉으로 스마트폰의 요소들을 이용하여 포토폴리오를 제작했습니다. 해상도 1920 * 1080에 최적화되어 있습니다. 스크롤 이벤트를 통해 원스크롤 기능이 구현되었으며 각 페이지 마다 아이폰의 테마들이 담겨져 있습니다.
